# App Legal Files

Centralized repository for managing mobile app legal files, including a single `app-ads.txt` for all apps under a Google AdMob account and separate privacy policies for each app. Designed for easy hosting via GitHub Pages or any static site provider, ensuring compliance with ad network and app store requirements.

---

## Repository Structure

```
my-app-legal-site/
├─ app-ads.txt                  # Single file listing authorized AdMob accounts
├─ privacy/                     # Folder containing app-specific privacy policies
│   ├─ app_one.html
│   ├─ app_two.html
│   ├─ app_three.html
├─ README.md                     # This file
└─ assets/ (optional)           # Optional folder for images or other assets
```
